1. LOGIN
$ curl -X POST http://localhost:3000/auth/login \
-H "Content-Type: application/json" \
--data '{"email":"rahulkakkar1104@gmail.com","password":"123"}'

{"message":"OTP generated","loginSessionId":"ow3cidtdo4r"}


2. VERIFY OTP
$ curl -c cookies.txt -X POST http://localhost:3000/auth/verify-otp \
-H "Content-Type: application/json" \
--data '{"loginSessionId":"ow3cidtdo4r","otp":"136185"}'

{"message":"OTP verified"}


3. GET TOKEN
$ curl -b cookies.txt -X POST http://localhost:3000/auth/token

{"access_token":"eyJhbGciOiJIUzI1NiIs...","expires_in":900}

$ curl -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..." \
http://localhost:3000/protected

{"message":"Access granted","user":{"email":"rahulkakkar1104@gmail.com"},"success_flag":"FLAG-..."}
